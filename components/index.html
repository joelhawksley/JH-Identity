Rethinking the view layer with components

The Rails view layer has been pretty stable since Rails 1.0 was released in 2005.

Recently, React and other frontend Javascript frameworks have risen to prominence in the Rails community, offering architectural advantages not found in Rails.

At GitHub, we've chosen to architect our frontend for progressive enhancement, which means that frameworks like React are off the table, as we need our app to work without Javascript enabled.
